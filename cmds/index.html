Admin Section
This section can only be used by moderators or administrators
This prevents the following commands from being used by normal users

kick [mention] [(opt) reason] - Kicks the mentioned user
ban [mention] [(opt) reason] - Bans the mentioned user
warn [mention] [(opt) reason] - Warns the mentioned user
timeout [mention] [time (sec)] [(opt) reason] - Timeouts the mentioned user
The command list above will be updated with new commands.
Mystique is currently in development and will be updated frequently.
